Func<Waveform, Bitmap> BenfWaves.Library.RenderedFormat.WaveformToBitmap
static

A delegate that must be initialized before attempting to save any images.

It accepts a waveform and returns a rendered bitmap. It's done this way to keep the format code in the library but decouple the UI, since rendering is done in the UI.
