enum BenfWaves.Library.TriggerMode
strong

The enumeration of all trigger modes possible for a profile.

Enumerator
NORM 

Continunously search for a trigger and don't update the display unless a trigger is found

AUTO 

Search for a trigger, forcing a trigger after 100ms if no trigger is found.

SING 

Continuously search for a trigger; do not initiate a new acquisition cycle after a successful trigger.

SCAN 

"In SCAN mode, data acquisition runs continuously and the input signal is progressively displayed from left to right in real-time disregarding other trigger options.

"

FIT 

"In FIT trigger mode, the Nano will automatically identify the type of waveform and adjust the settings to produce a usable display of the input signal.

"
